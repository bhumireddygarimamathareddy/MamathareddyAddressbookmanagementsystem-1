🚀 Address Book Management System in C

📞 A simple yet powerful console-based application to manage contacts efficiently using C programming and file handling.

✨ Project Highlights

🔹 Manage contacts with ease
🔹 Store data permanently using file handling
🔹 Modular and clean code structure
🔹 Beginner-friendly real-world project

🎯 Features

✔ Add new contacts
✔ Search contacts instantly
✔ Edit existing details
✔ Delete unwanted contacts
✔ Display all contacts
✔ Save & Load data from file
